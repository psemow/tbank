# Либа для отбора

Консольное приложение для управления личной коллекцией книг.

## Функциональность

- Добавление книги с указанием названия, автора, жанра, года издания и описания.
- Просмотр всех книг с возможностью сортировки (по названию, автору, году, жанру) и фильтрации (по жанру или статусу прочтения).
- Поиск книг по ключевым словам в названии, авторе или описании.
- Добавление книг в избранное и удаление из него.
- Изменение статуса книги ("прочитана"/"не прочитана").
- Удаление книги из библиотеки.
- Просмотр статистики: общее количество, прочитанные, избранные, топ жанров и авторов.
- Автоматическое сохранение данных в файл `liba.json` при выходе и загрузка при запуске.

## Требования

- Python 3.6 или выше.
- Не требует установки дополнительных пакетов (используются только стандартные модули: `json`, `os`, `datetime`).

## Запуск

```bash
python task2.py
```

## Использование

После запуска отображается меню с цифровыми пунктами. Ввод соответствующей цифры выполняет выбранное действие. Программа запрашивает необходимые данные и выводит результаты в консоль.

Пример работы:
1. Выберите `1` для добавления книги, введите название, автора и т.д.
2. Выберите `2` для просмотра всех книг, укажите параметры сортировки/фильтрации.
3. Выберите `0` для выхода — данные автоматически сохранятся.

## Структура данных

Каждая книга представлена словарём с ключами: `id`, `title`, `author`, `genre`, `year`, `desc`, `status`, `fav`, `date`. Все книги хранятся в списке и сериализуются в JSON.
